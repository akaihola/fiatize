What is this?

It's a cryptocurrency address balance viewer which also shows the balance as images of notes and coins of the selected fiat currency.

Bitcoin, Litecoin, Euro and US Dollar are currently supported. A QR code for the address can be scanned using a laptop's camera. The code works completely client-side in the browser.

Why do I need it?

I created it for my kids so they can see the balance of their Bitcoin allowance address in a more familiar form.

How are exchange rates determined?

The Bitcoin rate is checked from bitcoinaverage.com. The CryptoCoinCharts v2 API is used for Litecoin.
